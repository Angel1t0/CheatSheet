By G. Ángel Martínez

Introduction

Esta es una hoja de referencia para HTML y CSS. Es una compilación de las etiquetas y propiedades más comunes utilizadas en el desarrollo web.

HTML

Estructura Básica

HTML es un lenguaje de marcado que se compone de etiquetas. La estructura básica de un documento HTML incluye las etiquetas <html>, <head> y <body>. La etiqueta <html> envuelve todo el contenido del documento, <head> contiene metadatos y enlaces a archivos externos, y <body> contiene el contenido visible en el navegador.

Etiquetas de encabezado

Las etiquetas de encabezado se utilizan para definir los encabezados de una página. Hay seis niveles de encabezado, que van desde <h1> hasta <h6>. <h1> es el encabezado más grande y <h6> es el más pequeño.

Paragráfos y saltos de línea

Las etiquetas <p> se utilizan para definir párrafos. Las etiquetas <br> se utilizan para crear saltos de línea dentro de un párrafo.

Enlaces

Las etiquetas <a> se utilizan para crear enlaces. El atributo href se utiliza para definir la URL a la que se enlaza. El atributo target se utiliza para definir si el enlace se abre en una nueva ventana o en la misma ventana.

Listas ordenadas y no ordenadas

Las etiquetas <ul> se utilizan para crear listas no ordenadas. Las etiquetas <ol> se utilizan para crear listas ordenadas. Las etiquetas <li> se utilizan para definir los elementos de la lista.

Imágenes

Las etiquetas <img> se utilizan para insertar imágenes. El atributo src se utiliza para definir la URL de la imagen. El atributo alt se utiliza para definir el texto alternativo de la imagen, que se utiliza para describir la imagen cuando no se puede mostrar.

Tablas

Las etiquetas <table> se utilizan para crear tablas. Las etiquetas <tr> se utilizan para definir las filas de la tabla. Las etiquetas <td> se utilizan para definir las celdas de la tabla. Las etiquetas <th> se utilizan para definir las celdas de encabezado de la tabla.

Formularios

Las etiquetas <form> se utilizan para crear formularios. El atributo action se utiliza para definir la URL a la que se envían los datos del formulario. El atributo method se utiliza para definir el método de envío de los datos del formulario. Los métodos GET y POST son los más utilizados.

Etiquetas semánticas

Las etiquetas semánticas se utilizan para definir el significado de los elementos de una página. Las etiquetas semánticas incluyen <header>, <nav>, <main>, <section>, <article>, <aside> y <footer>.

Etiqueta Nombre Descripción

<img> Imagen Muestra una imagen dentro del documento.

<ol> Lista ordenada Define una lista ordenada con un orden numérico o alfabético.
<li> Elemento de lista Define un elemento de una lista.
<table> Tabla Crea una tabla para mostrar datos en filas y columnas.
<tr> Fila Representa una fila en una tabla.
<th> Celda de encabezado Representa un encabezado de columna o fila en una tabla.
<td> Celda de datos Representa una celda de datos en una tabla.

CSS

CSS es el lenguaje de hojas de estilo utilizado para dar estilo a la presentación de documentos HTML. CSS describe cómo se deben mostrar los elementos HTML.

Selectores

Los selectores CSS se utilizan para seleccionar los elementos HTML a los que se aplicarán los estilos. Los selectores CSS se pueden agrupar, anidar y combinar.

Propiedades

Las propiedades CSS se utilizan para definir los estilos de los elementos HTML. Las propiedades CSS se pueden agrupar, anidar y combinar.

Valores

Los valores CSS se utilizan para definir los valores de las propiedades CSS. Los valores CSS se pueden agrupar, anidar y combinar.

Propiedad Descripción

height Define la altura de un elemento.
width Define el ancho de un elemento.
margin Define el margen de un elemento.
margin-bottom Define el margen inferior de un elemento.
margin-left Define el margen izquierdo de un elemento.
margin-right Define el margen derecho de un elemento.
margin-top Define el margen superior de un elemento.
padding Define el relleno de un elemento.
